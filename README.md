## 📦 项目文件

| 文件名 | 类型 | 说明 |
|--------|------|------|
| `auto-exam.user.js` | 🤖 答题脚本 | **题库自动答题** - 加载JSON题库，智能匹配选项自动答题 |
| `collect-v3.user.js` | 📋 采集脚本 | **历史试卷采集** - 从已完成试卷中逐题采集参考答案 |
| `merged_questions.json` | 📚 题库文件 | **完整题库** - 459道题，包含单选、多选、情景分析题 |
| `merge_json.py` | 🐍 工具脚本 | 合并多个JSON题库并去重 |

## 🚀 快速开始

### 1️⃣ 安装 Tampermonkey

| 浏览器 | 安装链接 |
|--------|----------|
| Chrome | [Chrome 网上应用店](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) |
| Firefox | [Firefox 附加组件](https://addons.mozilla.org/firefox/addon/tampermonkey/) |
| Edge | [Edge 扩展商店](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd) |

### 2️⃣ 安装脚本

1. 点击 Tampermonkey 图标 → **添加新脚本**
2. 复制对应脚本文件内容粘贴到编辑器
3. 按 `Ctrl/Cmd + S` 保存

### 3️⃣ 开始使用

- **自动答题**: 进入考试页面 → 加载题库JSON → 开始答题
- **采集答案**: 进入历史试卷页面 → 开始采集 → 导出JSON

## 📖 使用说明

### 🤖 自动答题流程

```
1. 登录奥派直播电商运营实训系统
          ↓
2. 进入「测评中心」→「模拟考场」→「新建试卷」
          ↓
3. 页面右上角出现控制面板
          ↓
4. 点击「📁 加载题库JSON」选择 merged_questions.json
          ↓
5. 点击「▶ 开始答题」
          ↓
6. 等待自动完成，点击交卷
```

### 📋 答案采集流程

```
1. 登录系统，进入「测评中心」→「模拟考场」
          ↓
2. 找到已完成的试卷，点击「查看试卷」
          ↓
3. 页面右上角出现采集面板
          ↓
4. 点击「🚀 开始采集」
          ↓
5. 等待采集完成，点击「💾 导出JSON」
```

## 📚 题库格式

题库文件为标准 JSON 格式：

```json
{
  "exportTime": "2026/1/18 17:58:00",
  "totalQuestions": 459,
  "answers": [
    {
      "questionNum": 1,
      "questionType": "单选题",
      "options": ["A.选项1", "B.选项2", "C.选项3", "D.选项4"],
      "answer": "B"
    },
    {
      "questionNum": 2,
      "questionType": "多选题",
      "options": ["A.选项1", "B.选项2", "C.选项3", "D.选项4"],
      "answer": "ABCD"
    },
    {
      "questionNum": 61,
      "questionType": "情景分析题",
      "options": ["A.选项1", "B.选项2", "C.选项3", "D.选项4"],
      "answer": "1:A, 2:B, 3:ACD, 4:A, 5:BD"
    }
  ]
}
```

### 支持的题型

| 题型 | 答案格式 | 示例 |
|------|----------|------|
| 单选题 | 单个字母 | `"B"` |
| 多选题 | 多个字母连写 | `"ABCD"` |
| 情景分析题 | 小题编号:答案 | `"1:A, 2:B, 3:ACD"` |

## ❓ 常见问题

<details>
<summary><b>Q: 脚本加载后没有显示控制面板？</b></summary>

- 确认已正确安装 Tampermonkey 并启用脚本
- 确认访问的是正确的页面 URL
- 尝试刷新页面或清除浏览器缓存
</details>

<details>
<summary><b>Q: 答题时提示"未匹配"？</b></summary>

- 题库中可能没有该题，需要先采集补充题库
- 检查题库JSON格式是否正确
</details>

<details>
<summary><b>Q: 采集时答案全是同一个？</b></summary>

- 使用 V3 版本采集脚本 (`collect-v3.user.js`)
- V3 版本采用逐题点击等待的方式，确保正确获取
</details>

<details>
<summary><b>Q: 如何合并多个题库？</b></summary>

使用 `merge_json.py` 脚本：
```bash
python3 merge_json.py
```
会自动合并目录下的 1.json ~ 7.json 并去重
</details>
